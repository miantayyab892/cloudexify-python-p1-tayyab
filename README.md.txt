## Project Description
The **Personal Expense Tracker** is a Python-based Command Line Interface (CLI) application designed to help users efficiently manage their daily expenditures. It allows users to log expenses, view accumulated transactions, track total spending, filter items by category, and delete specific logs using a unique identifier.

---

## Features Implemented
- **Add Expense:** Easily log a new expense with its amount, category, and descriptive details.
- **View Expenses:** Display a list of all recorded expenses sequentially.
- **View Total Spending:** Instantly calculate and view the sum of all recorded expenses.
- **Filter by Category:** Search and filter expenses dynamically using case-insensitive category terms.
- **Delete Expense:** Remove a specific expense entry securely using its unique ID.
- **Graceful Exit:** Safely close the application at any time.
